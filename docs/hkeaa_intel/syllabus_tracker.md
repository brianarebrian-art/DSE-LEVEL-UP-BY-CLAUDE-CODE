## 檢查週期

- **每 90 日**：逐科抽查 HKEAA「Subject Information」頁有無新版評核大綱（注意：HKEAA 網站封鎖程式抓取，**必須人手瀏覽**；EDB PDF 可程式抓取）
- **每年 9–10 月**：考試年度交界，全面重查（HKEAA 慣常於此時發布下一屆框架）
- 發現變更 → 使用 [change_alert_template.md](change_alert_template.md)，48 小時內提報並評估題庫影響
